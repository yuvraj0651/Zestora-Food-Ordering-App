# 🍔 Zestora - Food Ordering Web App

## 📌 Overview
A real-world food ordering application with complete flow from browsing menu to checkout using modern frontend technologies.

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS

## ✨ Features
- 🍕 Menu listing with categories
- 🛒 Cart system
- 💳 Checkout flow
- 🔄 Dynamic UI updates
- 📱 Mobile responsive design

## 🧠 Key Learnings
- Managing multi-step flows (cart → checkout)
- UI/UX for real-world applications
- State synchronization

- Real-time status updates
